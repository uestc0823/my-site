## 项目起源

一直想做一个有特色的个人网站，而不是千篇一律的模板。作为一个《集合啦！动物森友会》的玩家，我被游戏中那种温暖、自然的视觉风格深深吸引。

恰好发现了一个开源的动森风格 React 组件库 —— `animal-island-ui`，于是决定基于它来构建自己的个人网站。

## 技术选型

- **框架** —— React 18 + TypeScript
- **构建工具** —— Vite
- **UI 组件库** —— animal-island-ui
- **部署** —— GitHub Pages
- **路由** —— 自定义 hash 路由（轻量，无需额外依赖）

## 设计思路

### 色彩方案

延续动森的温暖色调：

- 背景色：`#F7F1E5`（羊皮纸色）
- 文字色：`#794f27`（暖棕色）
- 强调色：`#19c8b9`（青绿色）
- 卡片：多彩但饱和度适中

### 页面结构

首页采用 NookPhone 风格的导航，内页使用左侧边栏 + 右侧内容的经典布局。移动端则自动切换为顶部导航栏 + 抽屉式菜单。

### 组件使用

充分利用 animal-island-ui 提供的组件：

- `Card` —— 项目展示、博客文章卡片
- `Collapse` —— FAQ 折叠面板
- `Divider` —— 装饰性分隔线
- `Modal` —— 项目详情弹窗
- `Title` —— 带丝带效果的标题
- `Typewriter` —— 首页打字机动画

## 开发过程中的挑战

### 1. 移动端适配

动森风格的圆角、阴影在小屏幕上需要精心调整。特别是 NookPhone 组件在手机上的缩放比例，经过多次测试才找到合适的值。

### 2. 性能优化

首页的滚动背景动画和打字机效果需要流畅运行。通过以下方式优化：

- 背景图使用 WebP 格式
- 打字机效果使用 `requestAnimationFrame`
- 组件懒加载

### 3. 博客系统

最初博客只是硬编码的文章列表，后来升级为基于 Markdown 的动态博客系统，支持图文混排。

## 总结

这个项目让我在实践中学习了 React 生态的方方面面，也让我意识到：好的个人网站不仅是技术的展示，更是个人品味和创造力的体现。
